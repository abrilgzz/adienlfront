Convocatoria Nacional 2018

Estamos enviando la convocatoria para el Campeonato Nacional de Patinaje Artístico Sobre Hielo 2018. Así mismo, estamos adjuntando el formato de Plan de Contenido y la Carta para las Escuelas. A partir de este momento están abiertas las inscripciones en la página www.adienl.org.mx. Las inscripciones cerrarán el Miercoles 24 de Octubre a las 4pm. Patinadores: deben entregar ficha de Inscripción y pago Entrenadores: deben entregar Hojas de Elementos y CD´s (Utilizaremos los CDs que se usaron para el Estatal. En caso de que quieran cambiarlo deberán entregarlos junto con la inscripción).
